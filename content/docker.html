Mô phỏng này giữ cùng tinh thần giao diện Docker Desktop: điều hướng tính năng theo sidebar, hiển thị nội dung theo tab và tối ưu responsive cho desktop/mobile.

🎯 Thử Docker ngay trên máy — 3 bước đầu tiên

Bước 1: Chạy container đầu tiên

docker run hello-world
# Docker tự tải image hello-world từ Docker Hub
# rồi chạy container in ra thông báo chào mừng.

Bước 2: Viết Dockerfile cho app Python Flask

# Dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
EXPOSE 5000
CMD ["python", "app.py"]

Tại sao COPY requirements.txt trước? Docker cache từng layer — nếu code thay đổi nhưng dependencies không đổi, bước pip install sẽ dùng lại cache, giúp build nhanh hơn 10-20x.

Bước 3: Build và chạy

docker build -t my-flask-app .
docker run -p 5000:5000 my-flask-app
# Mở trình duyệt: http://localhost:5000

Bước tiếp theo: Tìm hiểu docker-compose.yml để chạy nhiều container cùng lúc (ví dụ: Flask + PostgreSQL + Redis). Đây là kỹ năng bắt buộc cho bất kỳ dự án backend nào trong môn Phát triển web kinh doanh.
